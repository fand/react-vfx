VFX-JS is a JavaScript library to add WebGL-powered effects to your website.
You can easily attach it to normal <img />, <video> elements etc.

Usage

Install via npm:

npm i @vfx-js/core

Then create VFX object in your script:


import { VFX } from '@vfx-js/core';

const vfx = new VFX();
vfx.add(imgElement, { shader: "rgbShift" });

This will be rendered as below.

Examples

Image


<img src="example.png" />
                    

vfx.add(imgElement, { shader: "rgbShift" });

Output

GIF


<img src="example.gif" />
                    

vfx.add(gifElement, { shader: "rainbow" });

Output

Video


<video src="example.mp4" autoplay loop muted/>
                    

vfx.add(videoElement, { shader: "halftone" });
